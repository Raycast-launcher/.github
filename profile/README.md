# Raycast — Portable Launcher with AI, Extensions & Window Management

## About Raycast

Raycast provides a complete launcher environment for users who want keyboard‑first productivity. Raycast combines AI commands, window management, clipboard history, snippets, and thousands of extensions. Raycast supports app launching, file search, system commands, and custom scripts. Raycast enables users to control everything from a single hotkey. Raycast runs on desktop platforms. Raycast is trusted by developers, designers, writers, project managers, and productivity enthusiasts worldwide.

## Quick Reference for Raycast

| Operation | Raycast Action |
|-----------|----------------|
| Open Raycast | Option+Space |
| Search apps | Type app name |
| Run system command | Type command (Empty Trash, Sleep, Lock) |
| Open window management | Type "Window" or Cmd+Option+Arrow |
| Open clipboard history | Cmd+Shift+V |
| Create new snippet | Type "Create Snippet" |
| Run AI command | Type AI + prompt |

| Show floating notes | Type "Float Note" |
